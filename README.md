# Project Recipes
This project is part of the Odin Project's Foundations course. It is a basic recipes website designed to practice and solidify core HTML skills.

# Skills Demonstrated
. HTML Structure building a clean website using proprer elements like headings, paragraphs, and lists.

. Linking implementing internal and external links to connect different pages within a project.

. Directory Management organizing project files and assets into a clean folder structure.

. Git/GitHub Workflow managing a local repository and syncing progress to a remote host.
